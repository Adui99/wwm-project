April 30 Optimizations and Fixes

2026-04-30 LATEST

Below are the latest news and improvements from Eastwind Pavilion. Check them out!

Patch Notes for April 30th
1.Fixed a localization display issue in the reward preview for the limited-time "Echo" prize draw. In several languages, the amount of Summer Breeze currency was incorrectly displayed as "1,200" and "1,200". This has been corrected to "1,200" and "1,888". Please note that this was a visual bug only; the actual rewards and their probabilities were not affected.
2. Fixed a related display issue in the draw history for the "Echo" event, where the amount of Summer Breeze obtained was displayed incorrectly in multiple languages. This bug only affected the text in the history log. The currency amount shown during the draw, the actual rewards received, and all event probabilities were correct.
3. Fixed an issue that blocked progress in the Campaign mission, "Sunken City Lake".
4. Fixed an issue in the "Half-Anniversary" event where the "Real or Fake Snow Lion" quest did not correctly award the "Aroma Token" currency upon completion. We also corrected display errors for the currency's description and the quest's refresh timer.
5. Fixed an issue in the "Let It Snow" event where players could be repeatedly pulled back to the starting point and become stuck on a loading screen.
6. Fixed an issue where purchased Nameplates would fail to unlock. If you were affected, please check your in-game mail for a replacement Nameplate.
7. Fixed an issue causing incorrect prices and purchase limits for some items in the Sect Shop.
8. Fixed an issue that prevented the Umbrella weapon from being used in Relaying.
9. Fixed an issue on mobile devices where the Pose menu UI was disappearing.
10. Fixed an issue in the "Painting Qinchuan: Final Chapter" event where rewards were not granted after completing the photo task. The missing rewards will be automatically sent to all affected players.

Experience Optimization
1. Improved the recommendation logic for Open World Signposts to prioritize messages in your set language and filter out low-quality or irrelevant content.
2. Added a "Dungeon Guide" button to the Hero's Realm and Sword Trial interfaces for quick access to text and image walkthroughs. You can also ask Windtail directly to pull up relevant guides.
3. Enhanced the resolution and clarity for several set icons, including Jadeware, Formbend, Ripple Step, Calmwaters, Veil of the Willow, Jadeclasp, and Ebonward; Improved the visual connection between set names and their icons; and fixed an issue where item names were being cut off in pop-up windows.
4. Optimized memory usage to reduce crashes and improve overall performance on the PS5.
5. Added detailed rule descriptions for various Wandering Paths modes.
6. Increased the character limit for the search bar, allowing for more complete entries when looking up character names or keywords.
7. Most in-game event times will now display according to your device's local time zone for easier tracking.
8. Refined shadow details in Bamboo Retreat during the beginner tutorial.
9. Optimized the description for obtaining Mystic Skill · Goosy Slide. 
10. New Feaure: Weekly Ban List. Visit the Arena page to view the list of permanently banned players from the previous week. 

Bug Fixes
1. Fixed a rare issue where the "Lumina Guide - Liangzhou" Quest would not complete after finishing two "Wandering Tales" in Liangzhou.
2. Fixed a PS5 issue where the controller focus would disappear after the Battle Pass reached Level 50.
3. Fixed instances where Chinese text was still appearing in certain UI menus.
4. Fixed a controller bug that prevented players from scrolling to select rewards beyond Level 52 in the Battle Pass.
5. Fixed various localization errors across multiple interfaces.
6. Fixed a text display bug in the News window when running at 4K resolution.
7. Fixed an issue during the "Campaign - Death of the Governor" Quest where players were unable to use the "Flowing Glow" to reach the Governor's Residence.
8. Fixed an issue where the server was incorrectly identified following the addition of the US West server. 
9. Fixed an issue in the Spotlight Mode interface where buttons overlapped and cannot be interacted.
10. Fixed the issue with the final Talent cannot be activated in “Starlit Canvas - Howlion's Trial”. Wanderers can now claim the Leaf of Attainment via in-game mail, activate the talent, and attempt the challenge again.
