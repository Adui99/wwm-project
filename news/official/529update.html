1.7 Version Patch Notes: Optimizations and Bug Fixes (June 5, 2026)

2026-06-05 LATEST

June 5, 2026 Fixes and Optimizations

1. Servers in Oceania, the Middle East, and South America are now fully available. We will continue monitoring regional performance and improving our infrastructure to provide better service worldwide. We welcome any feedback or suggestions from our Wanderers.

2. Improved controller input responsiveness when playing the Papercutting minigame.

3. Improved the task descriptions for certain quests in the Imperial Palace, including clearer information on unlock times.

4. Improved loading performance during scene transitions on mobile devices.

5. Fixed an issue where certain Oddities in the Imperial Palace map could not be collected.

6. Fixed an issue where certain NPCs in Hexi: Familiar Faces could not be befriended.

7. Fixed an issue where the Jade Fish count on the Season page displayed incorrectly. Note: This was a display error only. The actual amount received was not affected.

8. Fixed an issue where the accessory Nine-Tail Vows behaved abnormally under certain conditions.

9. Fixed an issue in the A Warrior's Journey event where rewards could not be claimed after completing certain tasks. Also improved the event notification indicator behavior.

10. Fixed an issue in the Skyward Bond event where certain UI elements displayed in Chinese instead of the correct language.

11. Fixed an issue where the Thunder Enlightenment buff in Chapter 12 of the Taiping Mausoleum Path Trial was not applying correctly.

12. Fixed a visual issue with the snow display on the Imperial Palace map on PS5.

13. Fixed an issue where certain badges, name cards, and chat bubbles were missing.

14. Fixed an issue where the Skill Theft location for the Martial Art Unfettered Rope Dart could not be entered.

15. Fixed an issue where the chat bubbles Golden Ripples Night - Illusion and Golden Ripples Night - Stamen could not be used after being obtained.

16. Fixed an issue in the Jianghu Legacy: Grand Nuo Ritual where the game would freeze after drumming.


 

May 29, 2026 Fixes and Optimizations

1. Fixed a display issue on the [Soaring Record] page. During the affected period, the page was viewable but rewards could not be claimed. This issue did not affect future participation or reward collection. For gameplay details and the official release schedule, please refer to future official announcements and in-game content.

2. Fixed an issue in [Skyward Bond] where some players spawned outside the preparation area during the [Boss: Ghost Master] encounter.

3. Fixed an issue where the Palace map assets failed to load on mobile for some Wanderers after completing the update.

4. Fixed an issue where the [Guild Hero's Realm] page appeared blank for some Wanderers.

5. Fixed multiple UI display errors across various Guild gameplay pages.

6. Fixed several model display errors in the Kaifeng area.

7. Fixed an issue where players using a controller could not move the cursor to select items on the Season Shop page.

8. Fixed an incorrect English translation on the [Jadeware] 4-piece set bonus.

9. Fixed an issue where the default keybind for Sheathe Weapon/Stroll (hold) was incorrect and could not be reassigned to Z. After this fix, the default keybind has been updated to Z or Alt+Z, and both inputs are now functional.

10. Fixed an issue where some Wanderers were unable to claim check-in rewards during the [Palace Unveiled] event.

11. Fixed an issue where the [Mystic Skill Breakthrough Box] was missing from certain Development Shops.

 

We have heard your ongoing feedback about large install sizes and patch downloads, and we want to share what we are currently working on.

We are rolling out a [Quick Download Package] option on select platforms to help reduce download times and ease update pressure. This feature is currently in development for Steam. On mobile, we are also actively exploring ways to reduce the overall install size through asset optimization.

Please note that technical solutions vary across platforms, and some improvements will need to be implemented gradually as development progresses. We will continue to monitor reports of update errors and oversized patches, and work to address them as quickly as possible.

We appreciate your patience and understanding.
